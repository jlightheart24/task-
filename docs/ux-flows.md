# UX Flows (Draft)

## Core Task Entry
- Click next line to add a task.
- Checkbox at start of line to mark complete.
- Editor-like interactions for speed.

## Views
- Priority view
- Due date view
- Created date view
- Calendar (day/week/month)

## Calendar Drag and Drop
- Drag task to new day/time to update due_date.
- Updates should be immediate and local-first.

## Conflicts
- LWW applied automatically.
- User can set notification level:
  - none
  - summary
  - immediate
